Introduction from the developer Hi! Welcome to my app Belay, follow these simple steps to get started and make the most out of your experience! 1 Channels Channels serve as rooms where messages and replies are posted. They can be created by any user, and the channel list is actively updated to include those created by others. To create a channel Click the {+ create channel} button and input the channel name into the box labeled {What do you want to say}. If left empty, a unique name will be automatically generated for it. To switch to another channel Click the channel in the channel list. To edit channel name Click the pencil icon, input new_name then click {update} button 2 Messages Messages can be replied for any times. Get all messages in a channel Switch to the channel you want to chat in, click the channel in the channel list. Post a new message Once you're in a channel, you can start posting messages for others to see and respond to. Just type your message and hit the {Post} button. 3 Replies Replies can't be replied recursively, and you may reply to any message any time, it will be shown below that message. subdirectory_arrow_right To reply to a message Click the message block(including the replies belong to it) you want to reply to, it will show a red subdirectory_arrow just like this block. Then, input your reply into the {Reply to the comment} input bar. Finally click {reply} button.
